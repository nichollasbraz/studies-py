A simple Python game created to practice programming fundamentals. The objective is to find a hidden bomb randomly placed on a grid by guessing its row and column coordinates.\
**Note:** The game interface is in **Portuguese (Brazil)**.

## Features
• Interactive menu\
• Customizable grid size **(rows and columns)**\
• Random bomb placement\
• Input validation\
• Continuous gameplay until the bomb is found\
• Option to start a new game or exit
## Run 
```bash 
python main.py
```
## Concepts
• Functions\
• Loops **(while)**\
• Conditional statements **(if, elif, else)**\
• Exception handling **(try/except)**\
• Random number generation **(random)**\
• User input validation\
• Program flow control
## Purpose

This project was developed as a programming exercise to improve Python skills, problem-solving abilities, and understanding of control structures, validation, and game logic.

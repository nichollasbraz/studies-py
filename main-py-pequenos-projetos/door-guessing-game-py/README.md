A simple Python door-selection game created to practice programming fundamentals and decision-making logic.\
**Note:** The game interface is in **Portuguese (Brazil)**.

## Features
• Interactive menu\
• Three-stage progression system\
• Multiple door choices\
• Win and lose conditions\
• Input validation\
• Exit option

## Run
```bash
python main.py
```

## Concepts
• Functions\
• Loops **(while)**\
• Conditional statements **(if, elif, else)**\
• Dictionaries\
• User input validation\
• Variables and program flow

## How It Works

The player must pass through **three stages** by choosing one of three doors.

* **Stage 1:** Doors 1 and 2 are correct.
* **Stage 2:** Doors 1 and 3 are correct.
* **Stage 3:** Only Door 3 leads to victory.

Choosing a wrong door at any stage ends the game and returns the player to the main menu.

## Purpose

This project was developed as a programming exercise to improve Python skills, logical reasoning, and the use of dictionaries for managing game rules and progression.

A simple Python game created to practice programming fundamentals. The objective is to survive three stages by choosing the correct doors and reaching the final room.

**Note:** The game interface is in **Portuguese (Brazil)**.

## Features

• Interactive menu\
• Three-stage progression system\
• Multiple winning and losing paths\
• Input validation\
• Victory and defeat conditions\
• Return to menu after each game\
• Simple command-line gameplay

## Run

```bash
python main.py
```

## Concepts Practiced

• Functions\
• Loops (**while**)\
• Conditional statements (**if, elif, else**)\
• Dictionaries\
• User input validation\
• Exception handling (**try/except**)\
• Program flow control\
• Game state management

## Purpose

This project was developed as a programming exercise to improve Python skills, logical thinking, and problem-solving abilities through a simple multi-stage decision game.

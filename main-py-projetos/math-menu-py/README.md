A simple command-line calculator developed in Python to practice menus, loops, exception handling, and user interaction.

**Note:** The program interface is in **Portuguese (Brazil)**.

## Features

• Interactive menu\
• Addition, subtraction and division\
• Input validation\
• Error handling with **try/except**\
• Option to change the current numbers\
• Terminal screen clearing\
• Simple command-line interface.

## Run

```bash
python main.py
```

## Concepts Practiced

• Functions\
• Loops (**while**)\
• Conditional statements (**if, elif, else**)\
• Exception handling (**try/except**)\
• User input validation\
• Variables and arithmetic operators\
• Terminal manipulation (**os.system**)\
• Menu-driven program design.

## Purpose

Developed as a programming exercise to reinforce Python fundamentals, focusing on control flow, exception handling, and building interactive terminal applications.

A simple Python prototype that simulates cash withdrawals from an ATM by calculating the minimum number of banknotes required for a given withdrawal amount.

**Note:** The program interface is in **Portuguese (Brazil)**.

## Features

• Calculate banknotes for a withdrawal\
• Supports R$50, R$20, R$10, R$5 and R$1 banknotes\
• Input validation\
• Repeat withdrawals without restarting the program\
• Simple command-line interface.

## Run

```bash
python main.py
```

## Concepts Practiced

• Functions\
• Loops (**while**)\
• Conditional statements (**if, elif, else**)\
• Variables and accumulators\
• Integer arithmetic\
• Input validation\
• Exception handling (**try/except**)\
• Algorithmic problem solving.

## Purpose

This project was developed as a programming exercise to practice conditional logic and repetition structures by simulating the cash withdrawal mechanism of an ATM. It also serves as the foundation for a future, more complete banking system project.

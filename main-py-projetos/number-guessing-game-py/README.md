A simple Python game created to practice programming fundamentals. The objective is to guess a randomly generated number within a limited number of attempts.

**Note:** The game interface is in **Portuguese (Brazil)**.

## Features

• Interactive menu\
• Customizable number range **(1–99)**\
• Random number generation\
• Limited number of attempts\
• Higher/lower hints after each incorrect guess\
• Input validation\
• Option to start a new game or exit.
## Run

```bash
python main.py
```

## Concepts

• Functions\
• Loops (**while**)\
• Conditional statements (**if, elif, else**)\
• Exception handling (**try/except**)\
• Random number generation (**random**)\
• User input validation\
• Program flow control.

## Purpose

This project was developed as a programming exercise to improve Python skills, logical thinking, and problem-solving abilities through a simple command-line game.

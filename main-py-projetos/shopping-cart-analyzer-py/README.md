A simple Python application that records product information and generates purchase statistics through an interactive command-line interface.

**Note:** The program interface is in **Portuguese (Brazil)**.

## Features

• Register products with name and price\
• Calculate total purchase amount\
• Count products costing more than **R$1000.00**\
• Identify the cheapest product\
• Input validation\
• Error handling with **try/except**\
• Simple command-line interface.

## Run

```bash
python main.py
```

## Concepts Practiced

• Functions\
• Loops (**while**)\
• Conditional statements (**if, elif, else**)\
• Lists\
• Dictionaries\
• Exception handling (**try/except**)\
• User input validation\
• Variables and accumulators\
• Basic data analysis.

## Purpose

This project was developed as a programming exercise to reinforce Python fundamentals, focusing on data structures, loops, exception handling, and basic statistical analysis using user-provided data.

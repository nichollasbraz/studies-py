## Projeto final

### Objetivo
O objetivo deste projeto é proporcionar a prática dos conceitos vistos na disciplina por meio do desenvolvimento de um jogo interativo.

A sua tarefa é implementar um jogo de batalha naval. Nesse jogo, um jogador deverá tentar encontrar as embarcações do inimigo escondidas em um tabuleiro com o número mínimo de jogadas.

### Descrição da atividade

A especificação do sistema encontra-se do documento *especificacao.pdf*, na pasta *docs*.
Um exemplo do uso do programa já implementado está no vídeo *demonstracao_batalha_naval.mp4*, na pasta docs.

Você deve replicar esse comportamento implementando os módulos *Exibicao.py*, *FrontEnd.py*, *Tabuleiro.py* e *Persistencia.py*.
